# AI STORE (aistore.club) 自动化审计报告

本仓库存储了 [aistore.club](https://aistore.club) 网站的多维度自动化审计报告，由 [TestersAI](https://testers.ai) 生成。

## 报告列表

| 文件 | 说明 |
|------|------|
| [`index.html`](./index.html) | 用户角色反馈 — 模拟不同类型用户的体验分析 |
| [`bug-report.html`](./bug-report.html) | Bug 检测 — 功能缺陷与异常报告 |
| [`content-seo-audit.html`](./content-seo-audit.html) | Skills 页面内容与 SEO 审计 |
| [`copywriting-seo-report.html`](./copywriting-seo-report.html) | 文案质量与 SEO 优化报告 |
| [`wiki-content-seo-audit.html`](./wiki-content-seo-audit.html) | Wiki 页面内容与 SEO 审计 |

所有报告均为可直接在浏览器中打开的 HTML 页面。
